# Bellabeat_case_study
The Bellabeat case study involves a data analysis scenario where you, as a junior data analyst on Bellabeat's marketing team, explore smart device fitness data to unlock growth opportunities. 
Bellabeat is a high-tech manufacturer of health-focused products for women. The co-founder, Urška Sršen, believes that analyzing fitness data can guide marketing strategies. Your task is to focus on one Bellabeat product, analyze user data, and derive insights to shape the company's marketing strategy. The case study follows the data analysis process, covering tasks such as asking questions, preparing data, processing, analyzing, sharing findings, and suggesting actionable recommendations. The ultimate goal is to present a portfolio-ready case study that demonstrates your analytical skills and provides valuable insights for Bellabeat's marketing strategy.
